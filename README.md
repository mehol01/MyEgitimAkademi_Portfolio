# MyEgitimAkademi_Portfolio Projesi
Bu web uygulaması ASP .NET MVC 5 tabanlıdır. Web sitesinde yöneticiler içeriği denetleyebilirler. Site tamamen dinamiktir. Db first yaklaşımı kullanılmıştır.

# Teknolojiler
* ASP .NET MVC
* Entity Framework
* SQL Server
